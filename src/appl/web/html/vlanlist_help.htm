Getting started with IEEE 802.1Q VLAN

VLAN ID indicates the VLAN ID to control.

Member Ports indicates the number of member ports of the VLAN. There are two color symbols for each port:

         Tag egress packet
         Untagged egress packet


Create New Vlan creates the VLAN.

Click on the hyperlink of each VLAN ID to change member state or remove VLAN.
