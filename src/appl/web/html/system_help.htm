Getting started with System Information

Device name shows the switch device name. Click the hyperlink of the device name to change device name.

Firmware Version displays the revision ID of the system. Click revision ID hyperlink to start firmware upgrade process. The device will be rebooted and the upgrade process will start in few seconds.

Build Date indicates the build date of this firmware.

MAC Address indicates the MAC address of the out-of band Ethernet interface.

<%x@FEATURE:ENABLE(#IPV6)%> IPv6 Address indicates all the IPv6 addresses of the system.

<%/x%> IPv4 Address indicates the IPv4 address of the system.

Subnet Mask indicates the subnet mask of the IP address.

Gateway indicates the IP address of the gateway for the remote manager.

Link Ports indicates the current linked up ports

<%x@FEATURE:ENABLE(#bonjour)%> Bonjour shows Bonjour status. Click the hyperlink of status to turn on/off the Bonjour service.

<%/x%>

Refresh retrieves all system information again.

Factory resets the configurations of all pages to default value.

Save downloads device configurations into a file.

Restore restores previous saved configuration into the device.

Reboot reset device.
