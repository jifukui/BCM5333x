Getting started with Mirror

Port mirroring monitors traffic from specific ports to a single monitor-to port. Ingress and/or egress traffic is copied from the mirroring port to the mirror-to port.

Mode enables or disables mirroring.

Mirror specifies an Ingress/Egress Mirror port to which ingress/egress traffic will be mirrored.

Mirror To specifies the mirrored-to port.

Apply applies the mirror setting to the system.
