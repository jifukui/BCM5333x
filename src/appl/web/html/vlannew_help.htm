Getting started with IEEE 802.1Q VLAN

VLAN ID indicates the VLAN ID to control.

Member Ports indicates the number of member ports of the VLAN. There are three symbols for each port:

  Empty indicates that this port is NOT a member of the VLAN.
  U indicates that this port is a member of the VLAN. When the packet leaves the member port, the VLAN tag is removed.
  T indicates that this port is a member of the VLAN. When the packet leaves the member port, the VLAN tag is added.

Create To create this VLAN from the system.

Cancel To cancel for creating VLAN.
